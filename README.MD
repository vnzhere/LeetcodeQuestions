## 📈 Learning Habits
1. **No Code First:** Always dry-run on paper for 15 minutes.
2. **Space Optimization:** Always ask, "Can I do this in $O(1)$ space using pointers?"
3. **The "Why" Rule:** Comments in the code must explain the *reasoning* for moving a pointer, not just what the pointer is doing.
